# Cube Labs 3D Vision

Cube Labs 3D is a mobile-first platform for solving, learning, practicing, and
competing with twisty puzzles. The public experience — interactive hero cube,
playable NxN cubes, solvers, Cube ID profiles, and challenges — is the product.
The administration platform exists to operate that product safely: managing
users, managed advertising, competition integrity, security, and content without
code deploys, while protecting player data and keeping public rankings
trustworthy.

Long-term direction and the full feature checklist live in
[`ROADMAP.md`](./ROADMAP.md); architecture in [`ARCHITECTURE.md`](./ARCHITECTURE.md).
This short note exists so the documentation index resolves to a real file; expand
it as product strategy is formalized.
