Add the first real user-account and database foundation to Cube Labs 3D so players can register, sign in, maintain profiles, save solve history, and later challenge friends.

## Security Notes

- The publishable Supabase key is intended for client-facing applications and is protected by Row-Level Security.
- The Supabase service-role key must never be placed in browser code or in a `NEXT_PUBLIC_` environment variable.
- All future public-schema tables must have Row-Level Security enabled before launch.
- Authorization must continue to verify ownership using the authenticated user ID.
